# 백로그 (작업 대기열)

> 우선순위 순으로 정렬. 작업 시작 시 `/gsd-task`를 실행하여 current.md로 이동.

---

## 작업 등록 형식

```
### [우선순위] 작업명
- **목표**: 무엇을 만들 것인가
- **입력**: 어떤 데이터/파일이 필요한가
- **출력**: 어떤 결과물이 나와야 하는가
- **예상 난이도**: 상/중/하
- **등록일**: YYYY-MM-DD
```

---

## 대기 중인 작업

### [P1] config.py + requirements.txt 생성
- **목표**: 전역 설정값 중앙 관리 + 의존성 파일
- **입력**: 없음
- **출력**: `src/config.py`, `requirements.txt`
- **예상 난이도**: 하
- **등록일**: 2026-03-03

### [P2] api_client.py 생성
- **목표**: 기업마당 + K-Startup 공공데이터 API 호출 모듈
- **입력**: `src/config.py`
- **출력**: `src/api_client.py`
- **예상 난이도**: 중
- **등록일**: 2026-03-03

### [P3] crawler.py 생성
- **목표**: 기업마당 + K-Startup HTML 크롤링 보완 모듈
- **입력**: `src/config.py`
- **출력**: `src/crawler.py`
- **예상 난이도**: 중
- **등록일**: 2026-03-03

### [P4] data_processor.py 생성
- **목표**: 수집 데이터 중복 제거 / 필터링 / 정렬 파이프라인
- **입력**: `src/config.py`
- **출력**: `src/data_processor.py`
- **예상 난이도**: 중
- **등록일**: 2026-03-03

### [P5] notifier.py 생성
- **목표**: 콘솔 출력 포맷팅 (요약 / 마감임박 강조 / 전체 목록)
- **입력**: `src/config.py`
- **출력**: `src/notifier.py`
- **예상 난이도**: 하
- **등록일**: 2026-03-03

### [P6] storage.py 생성
- **목표**: JSON 또는 Markdown 파일 저장 모듈
- **입력**: `src/config.py`
- **출력**: `src/storage.py`
- **예상 난이도**: 하
- **등록일**: 2026-03-03

### [P7] main.py 생성 + 통합 테스트
- **목표**: 전체 파이프라인 오케스트레이터 + 실행 검증
- **입력**: P2~P6 모든 모듈
- **출력**: `src/main.py`, `output/YYYY-MM-DD_programs.json`
- **예상 난이도**: 중
- **등록일**: 2026-03-03
